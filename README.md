## `$the .ux internet`

<br/>

<div align="center">
<img src="https://user-images.githubusercontent.com/74038190/212284100-561aa473-3905-4a80-b561-0d28506553ee.gif" width="680"/>
</div>

<br/>

I run `ns1.nchikt.site` as an **authoritative DNS server** for the `.ux` top-level domain — same architecture as Google's `.dev` and `.app`. Anyone who sets DNS to `100.51.216.254` gets full `.ux` resolution.

```bash
# mac — use the .ux internet
sudo mkdir -p /etc/resolver
echo "nameserver 100.51.216.254" | sudo tee /etc/resolver/ux
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
open http://nachiket.ux
```

```
built with:   Node.js  ·  SQLite  ·  AWS EC2  ·  Nginx  ·  PM2
              no ICANN  ·  no gatekeepers  ·  just DNS and an idea
```
